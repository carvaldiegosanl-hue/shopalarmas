# ShopAlarmas

Sitio web de **ShopAlarmas** — seguridad electrónica profesional en Colombia: cámaras CCTV, sistemas de alarma, GPS vehicular, instalación y soporte.

Página web estática (HTML + Tailwind vía CDN). El formulario de contacto usa [Formspree](https://formspree.io/) y los botones de contacto abren WhatsApp.

## Publicación

Publicado con **GitHub Pages** desde la rama `main`.

## Desarrollo

No requiere instalación ni build. Abre `index.html` en el navegador.
